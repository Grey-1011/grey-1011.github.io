JS数据类型

  • 目前JS有七种数据类型：’四基两空一对象’
  • 分别是：number,string,boolean,symbol(符号)，null,undefined,object
  • 【注意】数组、函数、日期，他们都属于object,不是数据类型

Number: 64位浮点数

  • JS用64位浮点数存储Number

  • 特殊值：
  1. 正0 和 负0，正infinity 和 负infinity
  2. NaN 是无法表示的数字，它是一个数字
  • Number 范围和精度

String

  • 【注意】在JS中只有对象才有属性，但为什么字符串有属性？？？

  • 答： 在JS中，字符串会自动转化为字符串对象，所有字符串会有很多方法，比如split

  • 字符串的长度

  1. 字符串 \n\t\r 的长度是3
  2. ‘’ 空字符串的长度为0
  3. ‘ ‘ 空格字符串的长度为1

Boolean 布尔值

  • if 配 boolean,if 在判断真假时，如果value 不是布尔值，如何判断真假？
  • 答： 五个falsy值（既相当于false,又不是false的值）：undefined,null,0,NaN, ‘’,

Undefined 和 Null 两种空类型

  • 没有本质区别
  • 如果一个变量声明了，但没赋值，它的值就是undefined
  • 如果一个函数，没有写return,那么默认值就是undefined
  • 前端程序员习惯上，把非对象的空值写为undefined,把对象的空值写为null

类型转换

  • number => string
  1. number + ‘’
  2. String(number)
  • String => number
  1. Number(string)
  2. string - 0
  3. parseInt(string)/parseFloat(s)
  • x => bool
  1. Boolean(x)
  2. !!x
  • x => string
  1. string(x)
  2. x.toString()
