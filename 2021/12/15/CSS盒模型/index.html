简述盒模型

盒模型

浏览器会根据 CSS基础框盒模型(CSS basic box model) , 将所有元素表示为一个个矩形的盒子，CSS决定这些盒子的大小、位置以及属性。

每个盒子有四个部分组成:

  • content area
  • padding area
  • border area
  • margin area

两种盒模型：

  1. content-box width = content width
  2. border-box width = (content + padding + border)width

注意：外边距 margin重叠现象有时会发生

margin 合并问题

  • block元素，在 垂直方向 会发生合并margin现象，其大小为单个margin的最大值(或如果他们相等，则仅为其中一个)

    注意：如果有设定 浮动float 和 position=absolute，不会产生外边距重叠行为

  • 如何阻止合并

  1. 父子合并用 padding/border
  2. 父子合并用 overflow:hidden
  3. 父子合并用 display:flex
