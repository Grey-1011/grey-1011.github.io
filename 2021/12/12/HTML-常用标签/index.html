HTML重难点标签

常用标签-a

a标签： 属性(href / target / download / rel=noopener) & 作用

属性：

  • href –hyper reference
  • target
  • download
  • rel=noopener

作用：

  1. 跳转外部页面
  2. 跳转内部锚点
  3. 跳转到邮箱或电话等

a 的 href 取值：

  1. 网址
  • https://google.com
  • http://google.com
  • //google.com(无协议网址，会自动跳转选择协议，容错机制)
  1. 路径
  • /a/b/c (绝对路径，http服务的根目录)
  • a/b/c (相对路径)
  • index.html 以及 ./index.html (常用的方法)
  1. 伪协议
  • javascript: ; (用于点击后，不刷新，不跳转页面)
  • mailto:邮箱
  • tel:手机号
  1. id
  • href=”xxx” (跳转到指定标签)

a 的 target 取值：

  1. 内置名字
  • _blank (在新页面打开)
  • _top (在所有frame 最高层级上打开)
  • _parent (在当前链接所在层级的上一级打开)
  • _self (默认值，在自身页面打开)
  1. 程序员命名
  • window 的 name (利用同一个窗口打开页面，target=”xxx”,如果该窗口不存在，会自动创建并打开)
  • iframe 的 name (在同一个内嵌窗口下打开不同页面)

a 的 download：

  • 作用：下载页面
  • 问题：很多浏览器(手机浏览器)不支持

常用标签-img

作用

  • 发出get请求，展示一张图片

    属性

  • alt / height / width / src

    事件

  • onload/onerror 如果图片加载失败，触发事件，更换src

    响应式

  • max-width:100%

    可替换元素

  • 面试会问概率30%

常用标签-form

作用

  • 发get或post请求,然后刷新页面

    属性

  • action / autocomplete / method / target
  • action:要请求页面的的地址
  • autocomplete: 自动填充
  • method: 控制用 get 还是 post 请求
  • target: 控制提交到哪个页面

    事件

  • onsubmit: 要触发submit事件，必须要有type=”submit”

常用标签-input

作用

  • 让用户输入内容

    属性

    事件

  • onchange / onfocus / onblur
  • onchange: 当内容发生改变的时候触发
  • onfocus: 获得焦点时触发
  • onblur: 失去焦点时触发

    HTML5验证器

  • 例如：<input type=”text” required 当内容为空时，就会提示必填项，不能为空

其他输入标签

标签

  • select + option
  • textarea
  • label

    注意事项

  • 一般不 监听input 的click 事件
  • form 里面的input 要有name
  • form 里面要有type=”submit” 才能触发 提交事件
