HTML入门笔记

Created2021-12-11|Updated2021-12-19
|Post View:

HTML发明者—Tim Berners-Lee 李爵士

李爵士还发明了什么呢

  1. HTTP
  2. URL
  3. WWW = URL + HTTP + HTML
  4. WWW就是万维网 World Wide Web

HTML 起手应该写什么

感叹号+Tab

常用的表示章节的标签

  • 标题 h1~h6
  • 章节 section
  • 文章 article
  • 段落 p
  • 主要内容 main
  • 旁支内容 aside
  • 头部 header
  • 脚部 footer
  • 划分 div

全局属性有哪些

  • class 类
  • id
  • hidden 隐藏
  • style
  • tabindex 控制Tab键 属性和顺序，属性值： 0(最后访问) 正数(顺序访问) -1(不访问)
  • contenteditable 可以直接编辑标签内容
  • title

常用的内容标签有哪些

  • pre 保留空格效果
  • ol + li (ordered list + list item) 有序列表
  • ul + li (undered list + list item) 无序列表
  • dl + dt + dd (description list + term + data) 自定义列表
  • hr 分隔线
  • br 换行
  • a (anchor) 超链接
  • em (emphasis) 强调语气
  • strong 强调内容本身
  • code 用来包裹代码-等宽字体
  • quote 内联引用，不换行
  • blockquoye 块级引用
