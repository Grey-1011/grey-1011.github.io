CSS基础

Created2021-12-19|Updated2021-12-19
|Post View:

CSS牛X之处–在于层叠样式表，其中包括：

  • 样式层叠：可以多次对同一选择器进行样式声明
  • 选择器层叠：可以用不同选择器对同一个元进行样式声明
  • 文件层叠：可以用多个文件进行层叠

CSS的版本

  • CSS 2.1 ：是使用最广泛的版本(IE支持)
  • CSS 3 ： 是现代版本，分模块(IE8 部分支持)，之后就分模块升级

caniuse.com–查询浏览器兼容性

  • 可以在caniuse.com 上查询浏览器兼容哪些css特性

CSS语法

语法一：

选择器 {
属性名：属性值;
/* 注释 */
}

语法二：@语法

【注意事项】

  1. @charset 必须放在第一行
  2. 前两个 @ 语法必须以分号; 结尾
  3. charset 确定的是文件编码，而不是字符集

CSS的文档流–是指元素的流动方向

流动方向：

  • 当前Display:inline; 的元素从左到右，到达最右边才会换行，若宽带不够，会将自己截断，换行显示
  • 当前 Display：block; 的元素从上到下，每一个都另起一行
  • 当前 Display：inline-block，也是从左到右，但在一行末尾，不会像inline元素那样，截断自己

宽度：

  • inline 宽度为内部inline元素的和，不能用width 指定
  • block 默认自动计算宽度，可用width 指定 (width: auto;)
  • inline-block 结合前两者特点，可用width

高度：

  • inline 高度由 line-height 间接确定，也和字体有关，跟height 无关。（虽然增加padding,可以增加可视高度，但增加的并不是实际高度，如果span不写内容，还是有高度的，但是 div不写内容，高度为0）
  • block 高度由内部文档流元素决定，可以设height
  • inline-block 跟 block 类似，可以设置 height

overflow 溢出

  • 当内容大于容器
  1. 内容的width 或 height 大于容器的，就会溢出
  2. 可以用 overflow 来设置是否显示滚动条: 【注】overflow 可以分为 overflow-x 和 overflow-y

脱离文档流

  • 回忆一下：block元素高度由 ‘内部文档流元素’ 决定，可以设置height
  • 这句话也就意味着说，有些元素可以不在文档流中
  • 脱离文档流的方式:
  1. float
  2. position: absolute/fixed
