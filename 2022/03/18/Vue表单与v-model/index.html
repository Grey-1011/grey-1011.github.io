基本用法

文本 <input>

    多行文本 textarea 同上，区别不大

  • <textarea v-model="message" placeholder="edit me"></textarea>

  • 多个复选框

选择框

  • 使用v-for遍历
  • 多选时，加 multiple

form

  • 想要自动刷新页面，需要有一个 form 标签，且其标签里要有一个 button
  • 要想保存数据，需要在form上监听 submit

修饰符

.lazy:

  • v-model 默认监听 input 事件：键盘、鼠标等任何设备的输入
  • 常用的 change 事件，只在 input 失去焦点时触发

    .number

  • 当要求数据类型必须是 number 时，可使用 .number

    .trim

  • 自动去掉两头的空格

v-model

  • 当面试问 Vue的双向绑定，其实就是在问 v-model
  • 答：v-model 可以实现绑定一个变量，在变量变化的时候，UI会变化，用户改变UI的时候，数据也会变化，v-model实际上 是 v-bind:vale 和 v-on:input 的语法糖
  • 但是实际上 v-model 是 v-bind:value 和 v-on:input 的语法糖
  • 【注】v-on:input=”…”这里需要背下来
  • 自定义 input
