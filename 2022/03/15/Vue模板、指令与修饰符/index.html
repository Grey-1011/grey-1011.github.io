Vue模板、指令与修饰符

Created2022-03-15|Updated2022-03-16
|Post View:

模板 template三种写法

  • 一、 Vue完整版，写在HTML里
  • 二、 Vue完整版，写在选项里
  • 三、Vue非完整版，配合 xxx.vue文件

Vue模板里有哪些语法

展示内容:

表达式

  • {{ object.a }} 表达式 意思为 把构造选项的data里的object.a 显示到HTML里面
  • {{ n + 1 }} 可以写任何运算
  • {{ fn(n) }} 可以调用函数，会在 methods里找
  • 如果值为 undefined 或 null 就不显示
  • 另一种写法为 <div v-text="表达式"></div> 相当于 ，很少会用

HTML内容

  • 假设 data.x 值为 <strong>hi</strong> ，默认 显示的时候，就是 <strong>hi</strong>
  • 那该如何显示 粗体的 hi 呢？ 使用 <div v-html="x"></div>

就只是想展示 呢？

  • <div v-pre>{{ n }}</div>
  • v-pre 不会对模板进行编译， 就只是展示 字符

绑定属性:

绑定 src

  • <img v-bind:src="x" />
  • v-bind: 简写为：<img :src="x" />

    绑定对象

绑定事件:

v-on: 事件名

  • <button v-on:click="add">+1</button>
  • 点击之后，Vue会运行add()
  • <button v-on:click="xxx(1)">xxx</button>
  • 点击之后，Vue会运行xxx(1)
  • <button v-on:click="n+=1">xxx</button>
  • 点击之后，Vue会运行 n+=1
  • 即 发现函数就加括号调用之，否则就直接运行代码

缩写:

  • <button @click="add">+1</button>
  • 正常人都用缩写

条件判断: 什么时候出现在DOM 树里

  • if…else

    循环:

  • for(value,key) in 对象或数组
  • 【坑预警】:如果使用了 v-for 必须在后面加一个 :key=”index” ,否则有 bug

显示、隐藏:

v-show： 什么时候展示出来

  • <div v-show="n%2=== 0"> n是偶数 </div>
  • 如果v-show里的条件满足，就展示这个 div,否则就不展示
  • 近似等于
  • <div :style="{display:n%2===0?'block':'none'}"> n是偶数</div>
  • 但是要注意，看得见的元素display不只有block
  • 如： table 的 display 为 table
  • 如： li 的 display为 list-item

总结:

Vue模板主要特点:

  • 使用 XML语法 (不是 HTML)
  • <input name='username' />-必须使用/   <div/>如果标签为空，可以这样写
  • 使用 {{}} 插入表达式
  • 使用 v-html v-on v-bind 等指令操作DOM
  • 使用 v-if v-for v-show 等指令实现条件判断和循环

还没讲的指令

  • v-model 在Vue表单里讲
  • v-slot 在Vue 插槽里讲
  • v-cloak 看文档先https://cn.vuejs.org/v2/api/#v-cloak
  • v-once 看文档https://cn.vuejs.org/v2/api/#v-once

指令 Directive

什么是指令

  • <div v-text="x"></div>
  • <div v-html="x"></div>
  • 以 v- 开头的东西就是指令

语法：

  • v-指令名:参数=值，如 v-on:click=add
  • 如果值里没有特殊字符，则可以不加引号
  • 有些指令没有参数和值，如 v-pre
  • 有些指令没有值，如 v-on:click.prevent 阻止默认事件，如：
  • <a @click.prevent href="https://google.com">google</a>

修饰符

有些指令支持修饰符

  • @click.stop="add"表示阻止事件传播/冒泡
  • @click.prevent="add"表示阻止默认动作
  • @click.stop.prevent="add"既表示阻止事件传播，也表示阻止默认动作

一共有多少修饰符呢？

  • v-on 支持的有 .{keyCode | keyAlias}
  • .stop .prevent .capture(在捕获阶段监听) .self .once（只会触发一次的事件） .passive .native
  • 快捷键相关 .ctrl .alt .shift .meta .exact
  • 鼠标相关 .left .right .middle
  • v-bind 支持的有 .prop .camel .sync
  • v-model 支持的有 .lazy .number .trim

Vue修饰符 重点记忆

@click.stop="xxx" 阻止事件冒泡

  • 由于冒泡机制，给元素绑定点击事件的时候，也会触发父级元素的绑定事件
  • @click.stop可以阻止事件冒泡，相当于调用了 event.stopPropagation()方法

    @click.prevent="xxx" 阻止默认事件

  • @click.prevent用于阻止默认事件，相当于写 event.preventDefault()

    @click.stop.prevent="xxx" 既阻止冒泡事件，也阻止默认事件

    @keypress.event="xxx" 按键修饰符
