手写DOM库(对象风格)

  • DOM提供的API接口实在是不好用，于是手写了一个DOM库
  • 源代码链接：https://github.com/Grey-1011/DOM-1
  • 首先，我们提供一个全局的window.dom对象
  • 搭建环境：在html中，先引入dom.js,再引入main.js

增加节点

创建节点 dom.create(<div>newDiv</div>)

  • 使用方法

dom.after(node,node2) 新增弟弟

dom.before(node,node2) 新增哥哥

dom.append(parent.child) 新增儿子

dom.wrap(<div></div>) 新增爸爸

  • 使用方法

删除节点

dom.remove(node) 删除节点

dom.empty(parent) 删除后代

  • 如果用for 循环遍历删除后代，会有坑🕳
  • const {childNodes} = node 等价于 const childNodes = node.childNodes
  • 坑，就是： childNodes.length,每次都会实时变化，所以这里用while 循环

修改节点

dom.attr(node,name,value) 增加属性以及内容

  • 使用方法

dom.text(node,?) 读写文本内容

dom.html(node,?) 读写HTML内容

dom.style(node,{color:’red’}) 修改style

dom.class.add/remove/has(node,’blue’) 添加、删除、判断class

dom.on/off(node,’click’,fn) 添加、删除事件监听

  • 使用方法

查找节点

dom.find(‘选择器’) 获取标签或标签们

  • 使用方法

dom.parent/children(node) 获取父元素、子元素

dom.siblings(node) 获取同级元素

  • 找到父级元素的子元素
  • 过滤自己，条件是 n = node
  • Array.from转换伪数组
  • 调用的时候，还是加[0]

dom.next(node) 获取同级前面/后面的节点

dom.each(node,fn) 遍历所有节点

  • 使用方法

dom.index(node) 获取排行老几
